# Telco-Customer-Churn-Analysis-
Analysed telecom customer dataset using exploratory data analysis (EDA) to identify key churn drivers. showing low-engagement users had ~35–45% churn vs ~15–25% .Identified tech support as a critical retention factor, with churn ~45% (no support) vs ~15–20% (with support) .Proposed data driven recommendations that can reduce overall churn by ~20–25
